--- Mis competencias:

Habilidad Profesional

Profesionalmente como diseñador de UI/UX entiendo de la importancia de la información (Briefing) con respecto a los clientes para establecer las características del sitio web. La ejecución del proyecto por medio de la investigación y planificación para conocer el modelo de negocios al cual se apuntará . Crear un modelo visual o prototipo con diseño conceptual, (oportuna usabilidad web del sitio diseño centrado en el usuario) ayudado con programas de la familia ADOBE (Photoshop para manipulación de imágenes e Ilustrator para íconos, formas, figuras e imágenes en formato .svg), tengo preferencia por hacer el diseño responsive del sitio con Figma, para edición de video KDenlive, Ardour para el audio.


Profesionalmente como desarrollador poseo conocimientos de estructuración con HTML, tengo buenas bases en CSS y JavaScript, manejo de Visual Studio Code como editor de código fuente y un conocimiento básico de Visual Studio para desarrollo con C#. Me siento cómodo usando herramientas de uso común como Git, complementándolo con GitHub para alojar mis proyectos, Markdown para la documentación, y varias herramientas externas para agregar funcionalidades, optimización, ampliación, empaquetado y otros agregados más.

Las Soft Skills

Cómo desarrollador web Front-End soy consiente de la importancia de tener ciertas habilidades, tengo curiosidad para investigar el porqué de algunas cosas y comprender el correcto funcionamiento de estas, la puntualidad, conlleva la manera en que se deben realizar los trabajos, paciencia para comprender los problemas que se presenten y en consecuencia buscar soluciones. Ahora hablando también desde el diseño, marketing, hasta la gestión, como desarrollador sé de la importancia de la comunicación eficaz con multitud de departamentos para mantener el buen rumbo y completar los proyectos gracias al trabajo en equipo, finalmente, pude tener estudios en instituciones físicas y virtuales sin embargo mi conocimiento es también autodidacta viendo la documentación oficial, buscando soluciones alternativas, conociendo tecnologías nuevas y ahora probándome a mí mismo; que tan lejos puedo llegar en este campo.
